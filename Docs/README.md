# Documentation

This folder contains the core written documentation.  
These files describe the problem context, methodology, feature design, and
supporting analysis used in the project.

## Files Included

### Executive_Summary.docx
High-level overview of the project, including:
- Problem statement and objectives
- Methodological approach
- Key insights and outcomes

### Taxonomy.docx
Defines the conceptual and analytical taxonomy used for feature engineering:
- Risk dimensions and behavioral categories
- Alignment with established assessment frameworks
- Rationale for feature grouping and interpretation

### Prompt_for_LLM.docx
Details the prompt design used to guide Large Language Models:
- Structured prompt format
- Feature extraction instructions
- Consistency and reproducibility considerations

### Appendices.docx
Supplementary material supporting the main analysis:
- Additional tables and summaries
- Feature definitions and clarifications
- Extended methodological notes

## Notes
- All documents are intended for academic and educational use.
- No raw or sensitive data is included in this folder.
