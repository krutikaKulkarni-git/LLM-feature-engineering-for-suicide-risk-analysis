# Feature Schema

This folder contains the finalized feature schema produced as part of the
LLM-based feature engineering process used in this class project.

## Files Included

### Final_Feature_list.xlsx
Master list of engineered features derived from unstructured text:
- Feature names and categories
- Conceptual definitions and descriptions
- Logical grouping aligned with the project taxonomy
- Intended analytical interpretation of each feature

This file serves as the reference schema for all downstream analysis and
reporting discussed in the project documentation and presentations.
